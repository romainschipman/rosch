# Rosch-Button

The **`RoschButton`** component is a flexible and customizable button that integrates seamlessly with your application's theme. It can display custom content passed via the `children` prop, or fall back to using the `label` prop if `children` is not provided. The button also handles click events and can be disabled.

## Props

Here are the props that **`RoschButton`** supports, including `label`, `children`, `disabled`, and more, allowing for flexible customization.
